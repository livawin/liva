# Liva (LIVA)
Decentralized Prediction Markets on Solana

Liva (LIVA) is a utility and governance token powering the Liva decentralized prediction market ecosystem on Solana. It enables users to predict outcomes, stake tokens, and earn rewards across sports, crypto markets, and real-world events.
